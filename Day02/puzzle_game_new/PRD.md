# 프로젝트: 레벨 퍼즐 게임

## 목표
- 브라우저에서 실행되는 카드 매칭 퍼즐 게임을 만든다.
- 쉬운 난이도부터 시작해 레벨을 클리어할 때마다 다음 난이도로 자동 진행한다.
- 제한 시간 안에 퍼즐을 완료해야 하므로, 단순한 매칭 게임보다 긴장감 있는 플레이 경험을 제공한다.
- 실제 사용시스템처럼 폴더 구성도 하고, HTML, CSS, JavaScript를 독립적인 파일로 만든다.

## 핵심 경험
- 시작 화면에서 `시작하기` 버튼을 눌러 게임을 시작한다.
- 퍼즐을 맞추며 현재 난이도와 남은 시간을 확인한다.
- 한 레벨을 완료하면 다음 레벨로 자동 이동한다.
- 게임은 쉬움->중간1->중간2->중간3-> 어려움 순서로 진행된다.
- 모든 레벨을 완료하면 종료 화면에서 축하 메시지를 보여준다.
- 제한 시간 초과 시 게임오버 화면을 보여준다.

## 대상 사용자
- 브라우저에서 가볍게 즐길 수 있는 캐주얼 게임을 원하는 사용자
- 난이도와 제한 시간에 따라 단계적으로 도전하고 싶은 사용자

## 게임 규칙
- 같은 그림 2장을 찾아서 맞추는 메모리 게임이다.
- 카드 한 쌍이 맞으면 고정되고, 틀리면 다시 뒤집힌다.
- 시도 횟수는 카드 2장을 선택할 때마다 1회 증가한다.
- 각 레벨에는 제한 시간이 있으며 시간이 0이 되면 게임오버가 된다.
- 모든 카드를 맞추면 해당 레벨이 클리어된다.

## 레벨 구성
- 쉬움: `2x2`
- 중간1: `4x4`
- 중간2: `6x6`
- 중간3 : `8x8`
- 어려움: `10x10`

## 진행 방식
1. 게임을 처음 열면 시작 화면이 표시된다.
2. 사용자가 `시작하기` 버튼을 누르면 쉬움 난이도부터 시작한다.
3. 쉬움 레벨을 완료하면 중간1로 자동 이동한다.
4. 중간1, 중간2도 같은 방식으로 자동 진행한다.
5. 어려움 레벨까지 완료하면 최종 종료 화면을 보여준다.

## 화면 구성

### 1. 시작 화면
- 제목
- 간단한 설명
- `시작하기` 버튼
- 난이도 선택 콤보박스는 보여주되, 기본 시작은 항상 쉬움으로 시작한다.

### 2. 게임 화면
- 현재 난이도 선택 콤보박스
- 시도 횟수
- 남은 시간
- `다시 시작` 버튼
- 카드 보드
- 상태 메시지 영역

### 3. 클리어 화면
- 레벨 클리어 메시지
- 다음 난이도로 자동 이동 안내
- 마지막 레벨 완료 시 최종 축하 메시지

### 4. 게임오버 화면
- 시간 초과 메시지
- 다시 시작 유도

## 퍼즐 보드 규칙
- 난이도에 따라 보드 크기를 동적으로 생성한다.
- 카드 데이터는 emoji 또는 간단한 아이콘으로 구성한다.
- 짝이 맞는 카드만 고정한다.
- 짝이 안 맞으면 잠시 보여준 뒤 다시 닫는다.
- 홀수 칸수 보드(예: `3x3`, `5x5`)는 빈 칸 1개를 두어 UI 레이아웃을 유지한다.

## 시간 규칙
- 각 레벨마다 제한 시간을 둔다.
- 제한 시간은 난이도별로 다르게 설정한다.
- 시간이 0이 되면 입력을 막고 게임오버 상태로 전환한다.
- 게임오버 후에는 시작 화면으로 돌아가 다시 도전할 수 있다.

## UI/UX 요구사항
- 전체 레이아웃은 중앙 정렬 카드형 패널로 구성한다.
- 배경은 단색보다 부드러운 그라디언트와 분위기 있는 레이어를 사용한다.
- 주요 액션 버튼은 시각적으로 명확해야 한다.
- 모바일 화면에서도 보드와 버튼이 무너지지 않도록 반응형으로 구성한다.
- 상태 변화가 명확하게 보이도록 시작/클리어/게임오버 오버레이를 사용한다.

## 상태 관리
- `ready`: 시작 화면 상태
- `playing`: 게임 진행 중
- `cleared`: 현재 레벨을 완료한 상태
- `finished`: 모든 레벨을 완료한 상태
- `game_over`: 시간 초과로 종료된 상태

## 기술 스택
- HTML
- CSS
- JavaScript
- 각 기술별 파일 분리

## 구현 메모
- 단일 `index.html` 파일로 동작하도록 만든다.
- 외부 라이브러리 없이 구현한다.
- 카드 섞기는 Fisher-Yates 방식으로 처리한다.
- 게임 재시작 시 타이머와 선택 상태를 모두 초기화한다.
- 레벨 전환 시 현재 선택된 난이도 값을 다음 단계로 바꾼다.

## 완료 기준
- 시작 화면에서 게임이 시작된다.
- 쉬움부터 어려움까지 레벨이 순차적으로 진행된다.
- 제한 시간 초과 시 게임오버가 동작한다.
- 마지막 레벨 완료 시 종료 화면이 표시된다.
- 모바일과 데스크톱에서 모두 플레이 가능하다.
